# 小松久威のポートフォリオ

## 『MyV』の説明
自分の好きなAIと会話して動かせるアプリ「MyV」のコードです。
MyVは、ユーザーがイラストをアップロードすることで自由にAIの見た目を変更、
プロンプトを変えることで性格、音声も複数から選ぶことができます。
